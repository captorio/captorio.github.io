Captor

Captor is an HTTP protocol analyzer for Mac

System & Browser Configuration

In order for any application (or any browser) to use Captor it must be configured to use Captor as its proxy server.

When Captor is running, Captor toolbar shows both the listening IP addresses and port, like this:

Listening on 10.0.0.10:8080

Remember to disable the HTTP Proxy in your Settings when you stop using Captor, otherwise you'll get confusing network failures in your applications!

Mac OS X

  1. Click the Apple Menu.
  2. Click System Preferences.
  3. Click the Network icon.
  4. Click the Advanced button.
  5. Click the Proxies tab.
  6. Enable the Web Proxy (HTTP) and Secure Web Proxy (HTTPS) options to point to the Captor's IP address 10.0.0.10 (or 127.0.0.1 - same matchine as where the Captor is running), using port 8080 (default).

iOS Devices

  1. Tap Settings > Wi-Fi.
  2. Tap the settings (Information icon) for the Wi-Fi network.
  3. Tap the Manual option in the HTTP Proxy section.
  4. In the Server field, type the IP address of your Mac running Captor (example 10.0.0.10).
  5. In the Port field, type the port Captor is running (usually 8080).
  6. Leave Authentication set to Off.

Android Devices

  1. Swipe down from the top of the screen and tap the Settings icon.
  2. Tap Wi-Fi.
  3. Tap and hold your current Wi-Fi network. Select Modify Network.
  4. Tap to expand advanced options.
  5. Tap the Proxy settings dropdown and select Manual.
  6. Type the IP address (example 10.0.0.10) and port (usually 8080) of your Mac running Captor.
  7. Tap Save.

Windows

  1. Open Internet Explorer.
  2. Select Tools from the menu bar of your Internet Explorer windows.
  3. Select Internet Options
  4. Select the Connections Tab, and Click the LAN Settings button.
  5. Enable the Use a proxy server ... checkbox under Proxy Server. (the Address and Port edit boxes will now turn white and be editable).
  6. Enter the IP address (example 10.0.0.10) and port (usually 8080) of your Mac running Captor in their respective edit boxes.
  7. Click on OK and OK until all the windows have closed.

Firefox

  1. Click the Firefox menu. Hover over Options, and then select Options from the submenu.
  2. Open the Advanced category. Click on the Network tab.
  3. Click the Settings... button. This will open the Connection Settings window.
  4. Choose Manual proxy configuration. This will allow you to enter information in the fields below.
  5. Enter the IP address (example 10.0.0.10) and port (usually 8080) of your Mac running Captor in the HTTP Proxy field and Port field.
  6. Click on OK
